Media Player
This project is a smooth and interactive media player built using HTML, CSS, and JavaScript. 
The media player allows users to play songs, switch between tracks, adjust volume, and view dynamic album artwork for each song.

Features:
 Play and pause songs
 Next and previous song controls
 Dynamic album change for each track
 Volume up and down control
 Smooth user interface
 Responsive design

Technologies Used:
 HTML
 CSS
 JavaScript

This project demonstrates my frontend development skills and ability to create interactive web applications.
